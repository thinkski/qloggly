# QLoggly

Python interface to Loggly's Query API. Information on the API is available at
https://www.loggly.com/docs/api-retrieving-data/

## Credentials

Credentials are read from a `~/.loggly` configuration file that looks
something like:

    [loggly]
    account = acmecorp
    username = joe
    password = koolkatz
